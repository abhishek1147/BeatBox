PROJECT TITLE: BeatBox
PURPOSE OF PROJECT: The project was created to gain knowledge about the topics strictly.
VERSION or DATE: Version-1(beta),29-Aug-2017
HOW TO START THIS PROJECT: To start the project, directly run the source code provdied.
AUTHORS: Abhishek Tibrewal
LICENSE: All rights reserved to the author.
USER INSTRUCTIONS: The program offers the user to select beats from the ones listed and then as the user hits the "Start" button on the screen the program compiles all the beats selected by the user in the same order and plays them as a tune. User can also stop and make changes at any time and can also control the tempo of the tune by the buttons given.
